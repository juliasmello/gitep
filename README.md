### O que é Linux?

O Linux é um software utilizado por sua facilidade de implementação de códigos e adaptabilidade, além de robustez.

### O que é github?

Já o github é uma plataforma que permite o fácil compartilhamento de códigos e programas, além de possuir uma grande comunidade voltada para códigos abertos.
